# 🎮 Neon Defense - 霓虹防御射击游戏

一款策略射击游戏，你需要防御不断下降的数字蛇。管理资源，升级能力，在无尽的攻击中生存下来。

## 🎯 游戏特色

- 🎨 霓虹/赛博朋克风格视觉
- 🚀 流畅的射击体验
- 📈 动态难度系统
- 💎 战利品收集系统
- 🎁 能力升级系统

## 🕹️ 游戏玩法

1. **移动**：使用鼠标或触摸移动玩家
2. **射击**：自动射击，子弹会向上发射
3. **摧毁蛇段**：射击蛇的身体段来获得分数
4. **收集宝箱**：摧毁宝箱（带问号的方块）获得战利品
5. **选择门**：摧毁左右门之一来获得能力提升（攻击力或攻速）
6. **升级**：收集战利品提升攻击力、攻速或弹道数量
7. **生存**：不要让蛇的头部到达底部

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🛠️ 技术栈

- **React 19** - UI框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Canvas API** - 游戏渲染
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库
